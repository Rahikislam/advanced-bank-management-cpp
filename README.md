# 🏦 C++ Bank Management System

An advanced Bank Management System developed using C++ and Object-Oriented Programming (OOP).

## 🚀 Features
- Admin Login System
- User Registration & Login
- Deposit Money
- Withdraw Money
- Transfer Balance
- Transaction History
- Account Details
- Multiple Admin Support
- NID Verification System

## 🛠️ Technologies Used
- C++
- OOP Concepts
- Vector STL
- Console Application

- Inheritance
- Encapsulation
- Polymorphism
- Abstraction
